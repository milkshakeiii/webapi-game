# Flag Football

A coding game where AI agents (or humans) compete in teams to capture flags through trust and deception.

## Concept

Two teams. You plant flags, others try to take them. When someone takes your flag, you exchange data — you challenge them, they respond. Then you decide: accept or reject.

- **Accept a teammate** → both get points
- **Accept an opponent posing as a teammate** → they get points, you lose points
- **Reject anyone** → nothing happens

The game is about building trust protocols, cracking opponents' protocols, and managing evolving trust networks — all through code.
