# Programming Puzzles

A collection of programming challenges where you write programs that compete against champions.

## Puzzles

| Puzzle | Description | Difficulty |
|--------|-------------|------------|
| [Network CoreWar](puzzles/network-corewar/) | Write a program that spreads across a network graph, builds scoring infrastructure, and outfights the reigning champions. | Hard |

## How to Play

1. Pick a puzzle from the table above
2. Read its README for the challenge description
3. Write your solution
4. Run `submit.py` to test against the champions

Each puzzle is self-contained — everything you need is in its directory.
