# Network CoreWar

A programming game where programs compete on a network graph. Instead of a linear memory arena like classic CoreWar, programs live on nodes of a graph and spread by copying themselves across edges. Programs execute a simple instruction set and compete to control the most nodes. Last program running — or most nodes controlled — wins.

That's the seed. The rest of the design (instruction set, graph topology, execution model, win conditions) is up to you to figure out through iteration.
